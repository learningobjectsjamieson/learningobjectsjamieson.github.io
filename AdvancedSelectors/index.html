Understanding Advanced Selectors in CSS

Sometimes you want to select all tags in an HTML page..

However, most of the time you want to style only specific tags that you've coded in HTML. Selecting them all in your CSS is not necessarily going to be the best thing for you to do all the time.

As an example, maybe you want some paragraphs (like this one) to be the color purple. If you simply code a p as your selector in CSS, it will select every paragraph in your webpage.
